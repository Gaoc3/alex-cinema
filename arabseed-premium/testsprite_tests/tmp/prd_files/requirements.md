# AleX Cinema - Premium Streaming Platform

## Overview
Arabic streaming platform providing ad-free access to Arabic movies, series, and anime content via the Cinemana/Shabakaty Android API proxy.

## Features
1. Browse movies, series, and anime by category
2. Full-text search with advanced filters (genre, year, rating)
3. Video playback with multiple quality options and subtitles
4. Responsive RTL design supporting all devices

## Technical Requirements
- Dual frontend: Flask (legacy) + Next.js 16
- Video streaming via HLS with player support
- API proxy tunnel for CDN bypass
- Arabic-first UI with RTL layout

## Target Users
Arabic-speaking audiences seeking free, ad-free streaming of movies and series.
